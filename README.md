# Eksperimen_SML_Najwan_Mursyidan

Repository ini berisi eksperimen dataset pelatihan menggunakan Template Eksperimen MSML.

Struktur utama:
- `breast_cancer_raw.csv`: dataset raw.
- `preprocessing/Eksperimen_Najwan_Mursyidan.ipynb`: notebook eksperimen yang sudah dijalankan.
- `preprocessing/automate_Najwan_Mursyidan.py`: script otomatisasi preprocessing.
- `preprocessing/breast_cancer_preprocessing.csv`: dataset hasil preprocessing.

Cara menjalankan preprocessing:

```bash
cd preprocessing
python automate_Najwan_Mursyidan.py
```
